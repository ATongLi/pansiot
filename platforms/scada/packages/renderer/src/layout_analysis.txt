# 布局链分析

1. #root (height: 100vh, overflow: hidden)
2. .app (display: flex, flex-direction: column, height: 100%)
3. TitleBar (position: fixed, height: 42px) - 脱离文档流
4. .app-body (flex: 1, display: flex, padding-top: 42px, overflow: hidden)
5. .main-content (flex: 1, height: 100%, overflow: hidden)
6. .homepage (display: flex, width: 100%, height: 100%, overflow: hidden)
   ├── .sidebar (width: 60px, height: calc(100vh - 42px)) ❌ 问题！
   └── .homepage-content (flex: 1, overflow-y: auto)

问题分析：
- .app-body 有 padding-top: 42px，但实际高度是 flex: 1
- .main-content 是 height: 100%，继承 .app-body 的高度
- .homepage 是 height: 100%，继承 .main-content 的高度
- .sidebar 是 height: calc(100vh - 42px)，这会超出父容器！
