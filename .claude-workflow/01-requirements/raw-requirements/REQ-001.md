# REQ-001: Scada基础主页面框架

## 需求信息

**需求ID**: REQ-001
**需求标题**: Scada平台基础主页面框架
**需求来源**: 用户
**创建日期**: 2026-01-20
**目标平台**: Scada
**优先级**: 高 (P0)
**状态**: ✅ 已完成

## 需求描述

实现Scada平台的基础主页面框架，为工业物联网HMI工程编辑器提供用户界面基础。

## 功能需求

### 1. 整体风格要求
- **设计风格**: 工业物联网软件风格
- **设计原则**: 精简、专业、高效
- **参考原型**: `asset/组态软件框图-第 5 页.drawio.png`

### 2. 页面布局要求

#### 2.1 顶部栏 (Top Bar)
- **高度**: 40px
- **内容**:
  - 左侧显示应用标题 "PanTools"
  - 右侧显示窗口控制按钮（最小化、最大化、关闭）
- **交互**: 支持窗口拖拽（Electron集成）

#### 2.2 左侧导航栏 (Sidebar)
- **宽度**: 80px
- **导航项**: 5个（首页、本地、云端、工具、User）
- **交互**:
  - 点击切换激活状态
  - 激活项使用粉色/红色背景 (#FF9999)
  - 悬停效果

#### 2.3 主内容区 (Main Content)
- **操作按钮组**:
  - "从文件打开"
  - "新建工程"
  - "复制工程"
- **最近工程列表**:
  - 显示最近打开的工程
  - 卡片式布局
  - 悬停效果

## 技术约束

### 前端技术栈
- React 18.3.1
- TypeScript
- MobX 6.12.0 (状态管理)
- Vite 5.4.2 (构建工具)

### 样式方案
- Plain CSS (不使用CSS-in-JS)
- CSS Variables (主题系统)
- BEM命名规范

### Electron集成
- 窗口拖拽区域 (`-webkit-app-region: drag`)
- IPC通信准备 (当前为Mock实现)
